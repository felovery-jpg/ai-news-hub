# ai-news-hub

AI News Hub - Auto-collect AI news, tools & research daily.

## Features
- Daily auto-collect AI news from 16+ RSS sources
- AI-powered summarization with Gemini API
- SEO optimized with sitemap, robots.txt, structured data
- Ad monetization ready (Google AdSense + affiliate links)
- Automatic deployment via Vercel

## Tech Stack
- Next.js 14.2 (App Router)
- TypeScript + TailwindCSS
- Supabase (database)
- GitHub Actions (daily automation)
